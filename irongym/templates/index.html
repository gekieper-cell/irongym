GOOGLE DRIVE & LOOKER STUDIO
EXPORTAR DATOS

Exportá tus datos en formato CSV y subílos a Google Drive. Luego conectá Google Drive como fuente de datos en Looker Studio para crear dashboards personalizados.

INSTRUCCIONES — CONECTAR CON LOOKER STUDIO
  1. Exportá los archivos CSV usando los botones de arriba.
  2. Subí los archivos a una carpeta en tu Google Drive.
  3. Abrí Looker Studio → Nuevo Reporte → Agregar datos.
  4. Seleccioná Google Sheets o subí el CSV vía File Upload.
  5. Conectá el archivo y creá tus visualizaciones personalizadas.
  6. Para mantener los datos actualizados: exportá y sobreescribí el archivo periódicamente.
ACCESO DESDE CELULAR EN RED LOCAL
